# Deloitte Data Analytics Virtual Experience

This repository contains my solutions for Deloitte’s Data Analytics Virtual Experience Program, where I worked on transforming and standardizing real-world machine telemetry data.

---

## 📌 Project Objective

The goal of this project was to process raw data from different formats and convert it into a unified structure that can be used for analysis and dashboard visualization.

---

## ⚙️ Problem Statement

The system receives data from multiple devices in different formats:
- Format 1: Simple structured JSON
- Format 2: Nested JSON with timestamps

The task was to:
- Normalize both formats into a common structure
- Ensure consistency across all data
- Prepare data for further analysis and visualization

---

## 🛠️ Solution Approach

- Identified different input formats
- Created separate functions to handle each format
- Converted timestamps into a standardized format
- Structured location data into hierarchical fields
- Unified all outputs into a single consistent JSON structure

---

## 📂 Project Structure
- data-1.json → Sample input data (Format 1)
- data-2.json → Sample input data (Format 2)
- data-result.json → Final processed output
- main.py → Python script for data
